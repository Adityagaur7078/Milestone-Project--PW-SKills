## 🚀 Running the Bike Landing Page (Tailwind Project)

The **Bike Landing Page** uses Tailwind CSS, so the Tailwind build process must be running while viewing the project.

### 1️⃣ Navigate to the project folder

```bash
cd "Bike Landing Page"
```

### 2️⃣ Start the Tailwind build process

```bash
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

This command compiles Tailwind CSS and watches for changes.

### 3️⃣ Open the project

Open the following file in **VS Code** and run **Live Server**:

```
src/index.html
```

The webpage will open in your browser with Tailwind styles applied.
